# react-native-simple-jsi

How to build a jsi module example library

## Installation

```sh
npm install react-native-simple-jsi
```

## Usage

```js
import SimpleJsi from "react-native-simple-jsi";

// ...

const result = await SimpleJsi.multiply(3, 7);
```
